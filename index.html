Tips

Why this tool matters

When one tiny road segment is still missing, scanning the whole map by eye is painful. This tool highlights likely undiscovered-road pixels so you can focus on suspicious spots instead of checking every road again.

Compatibility and screenshot rules
  • Built for Forza Horizon 6.
  • FH1 to FH5 may not work correctly unless support is added later.
  • Use screenshots captured directly from the game.
  • Phone photos of a screen cannot be scanned by this tool because camera processing and screen glare change the colours.
  • PNG is best. JPEG can work if it is a direct screenshot, but may need tolerance 5 or 8.
How to take a clean map screenshot
  1. Open the Forza Horizon 6 map screen.
  2. Zoom out to see as much map as possible.
  3. Use PNG format (Win+Shift+S, paste into Paint, save as PNG).
  4. JPEG works if it is a direct screenshot, but may need tolerance 5 or 8.
  5. Avoid overlapping UI elements.
How to find small missing roads
  1. Switch to Black output mode.
  2. Increase Line thickness to make tiny dots visible.
  3. Scroll to zoom in, drag to pan around.
  4. Use 1:1 to see actual pixels.
  5. Try different highlight colours for contrast.
Tolerance guide
  • 0 — Exact match. Clean PNG only.
  • 1-2 — Strict/Normal. Good for PNG.
  • 5 — Loose. Start here for JPEG.
  • 8+ — Very loose. More false positives.

JPEG compression shifts pixel colours, so higher tolerance is needed.

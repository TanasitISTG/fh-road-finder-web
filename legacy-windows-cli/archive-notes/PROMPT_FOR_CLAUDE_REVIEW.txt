Please review this consolidated FH6 Road Finder build:

C:\Users\user\Desktop\fh6_road_finder\consolidated

Goal:
Combine the best parts of the Claude and Codex versions into one portable Windows tool for detecting possible missing roads in Forza Horizon 6 map screenshots.

Important current decision:
Grid mode was intentionally removed to keep the workflow simpler. Please do not re-add it unless you find a strong reason. Instead, check that no BAT prompt, README instruction, or Python CLI option still exposes grid mode.

CSV output was intentionally removed. The user only wants to visually inspect the generated images because screenshot coordinates are not useful to match in-game.

Tiny blob suppression was intentionally removed from the generated images. Overlay, black highlight, and transparent mask files should use the raw detected mask so even very small red hits remain visible. Any cluster counting in the console/report is only secondary and must not hide pixels in the images.
